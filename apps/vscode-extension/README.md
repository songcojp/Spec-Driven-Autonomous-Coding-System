# Spec Explore

Spec Explore is the VSCode-native SpecDrive workbench for browsing Spec documents, opening Feature Specs, and controlling execution queue state from the editor.

Use the Spec Explore activity bar entry to open the three core workbench pages:

- Spec Workspace
- Feature Spec
- Execution Workbench

Configure the local Control Plane and Product Console URLs through the `specdrive.controlPlaneUrl` and `specdrive.productConsoleUrl` settings.
